# iOS Debug Bridge 

iOS Debug Bridge (idb) is a versatile command-line tool that lets you communicate with a device.



## Query for devices

```
$ idb devices

```

## Push file into device

```
$ idb push test.txt /Temp

$ idb push test.txt /Temp  --bundle="com.seasun.tmgp.jx3m" -d = 1

```
